# Warp
Inspired from [CaryKH's video editor](https://www.youtube.com/watch?v=DQ8orIurGxw), it's an attempt to provide students the ability to watch lectures at an 'optimized' rate, by using Machine Learning and a custom FFmpeg plugin to warp a video based on the presence of speech or not.
