# Warp

Based on [https://www.youtube.com/watch?v=DQ8orIurGxw&t=205s](CaryKH's video editor), it's an attempt to provide students the ability to watch lectures at an 'optimized' rate, by using Machine Learning and a custom FFmpeg plugin to warp a video based on the presence of speech or not.
